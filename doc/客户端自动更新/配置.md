#### 客户端的自动更新

```
只更新webpack打包后的对应的静态资源文件(public目录下)
```

- 服务器

  - <PROJECT>/public 为静态资源文件
  - <PROJECT>/build.js 打包更新文件，生成更新文件与version.json版本文件
  
  - package.json
    
    - version 控制版本
    - author 控制发布人
    - description 控制详情
    

- 客户端
  - <PROJECT>/version.json 当前版本文件
  
  
过程
1. vue项目中打包,在server/public中生成app资源文件夹,在views中生成app.html或者app.ejs文件(不在更新范围之内)
2. server项目中配置 scripts:{"build": "node ./versions/build.js --v=1.0.0 --d=app,desk,*.js,*.css"} ,运行npm run build,生成versions文件夹下app资源文件与version.json文件.
3. client项目下更新(下载至publuc文件夹下)
